== Description ==

Display a single download as a static homepage when using Easy Digital Downloads.

== Installation ==

1. Install and Activate.
2. Create a dummy page that is not used anywhere else on the site.
3. Visit "Settings > Readings" and assign a static homepage to to the dummy page.
4. Pick the download you want in "Single Product Home" dropdown.
